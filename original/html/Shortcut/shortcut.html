Shortcut Function Introduction

This page introduces the shortcut key functions available on the canvas.

Shortcut Key List

Shortcut Keys Function
Ctrl + G Toggle grid display
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + C Copy( When Image is selected )
Ctrl + V When a panel is selected: Copy image to place the image inside the panel. When something other than a panel is selected: Copy image to place the image inside the Canvas.
Delete Delete selected layer
Arrow Keys Move selected object
Ctrl + L Toggle layer panel display
Ctrl + K Toggle controls display
Ctrl + 8 Zoom in canvas.
Ctrl + 9 Zoom out canvas.
Ctrl + 0 Zoom reset canvas.
